# hiss-vscode README
Experimental Hiss support for VSCode

## Features

* Syntax highlighting for .kiss files

## Requirements

* Haxe 4 installed and available in your system PATH.
* haxelibs:
    * hxnodejs
    * vscode
    * kiss
    * hscript
    * uuid
    * tink_macro
